# Getting the datasets
	1. The three datasets named CrimeClassification.csv, Forex_chfjpy and Skin_nonskin.txt  are available under the zipped file named Data.zip.
	2. The three code files required are Skin_NonSkinClassification.ipynb, CrimeIncidents.ipynb and ForexClassification.ipynb.

#Running the code
	To run the code please place the three datasets and the Jupyter notebook files in the same directory.
